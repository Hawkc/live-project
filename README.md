# 抽奖

为了鼓励大家积极进行有意义的发言，现举办一轮抽奖活动。

规则发言中加入关键词即可参与抽奖

## 关键词：
- #我要参与换组活动#
- #我要红包#
- #我支持调课的决定#
- #我要当学习委员#


## 模式选择：

- **不过滤模式**：参与抽奖的所有人
- **普通模式**：筛除只参与抽奖而无任何言论的用户（抽奖机器人）
- **深度模式**：对以下用户进行降权处理：

1. 只参与抽奖而无任何言论（抽奖机器人）
2. 只参与抽奖且只发送表情（水军）
